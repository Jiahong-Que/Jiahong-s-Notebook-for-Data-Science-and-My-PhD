## 🧹 如何清理 Mac 上的“系统数据”占用？

“系统数据”包括缓存、日志、临时文件、本地快照、磁盘映像等，这些通常无法直接在“访达”中看到，但可以通过以下步骤进行清理：

---

### ✅ 第一步：清空“废纸篓”

* 右键点击“废纸篓”图标，选择\*\*“清空废纸篓”\*\*

---

### ✅ 第二步：删除 Time Machine 本地快照（占空间大户）

打开“终端”（Terminal），输入以下命令查看本地快照：

```bash
tmutil listlocalsnapshots /
```

然后删除：

```bash
sudo tmutil deletelocalsnapshots <快照日期>
```

或一次性清理旧快照（例如清理10GB）：

```bash
sudo tmutil thinlocalsnapshots / 10000000000 4
```

---

### ✅ 第三步：清理缓存文件

打开“访达” > “前往” > “前往文件夹...”：

```bash
~/Library/Caches
```

删除每个子文件夹中的内容（**不要删除整个文件夹**）。

然后进入系统级缓存：

```bash
/Library/Caches
```

同样清理里面的内容。

---

### ✅ 第四步：删除旧的安装包（.dmg/.pkg/.iso）

在“访达”中按下 `Command + F`，搜索：

```
.dmg  .pkg  .iso
```

选择大文件、旧文件删除。

---

### ✅ 第五步：清理 iOS 设备备份

进入以下路径查看并删除旧备份：

```bash
~/Library/Application Support/MobileSync/Backup
```

---

### ✅ 第六步：删除不再使用的应用程序

你当前的**应用程序占用了 60.85 GB**，建议：

* 打开“访达” > “应用程序”，
* 查看哪些应用很大但不常用，右键删除。

---

### ✅ 第七步：清理系统日志文件

打开“访达” > “前往文件夹...”：

```bash
/var/log
```

删除旧的 `.log` 日志文件。

---

### ✅ 第八步：重建 Spotlight 索引（如果索引数据太大）

打开终端输入：

```bash
sudo mdutil -E /
```

---

### ✅ 第九步（可选）：使用清理工具

推荐工具：

* **CleanMyMac X**（付费，界面友好）
* **OnyX**（免费，功能强大，适合高级用户）

---

## 📌 额外提示：

* 清理完成后建议**重启电脑**，释放系统资源。
* **不要**手动删除 `/System`、`/usr`、`/Library` 等核心系统文件夹中的内容。
